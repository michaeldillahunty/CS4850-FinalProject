# CS4850-FinalProject

## Requirements
- Unix/Linux OS
- gcc 
- CMake (version > 2.8)

### Compiling 
Run the following commands:
- in `V1` directory: `mkdir build`
- `cd build`
- `cmake ..`
-  `make`
   - this will output 2 executable files [client and server]
- run each using `./client` or `./server`
